# hubot-idotaba

Hubot on Idobata

## Usage
* install forever 

        $ sudo npm install forever -g

* change directory

        $ cd hubot-idobata

* addnode_module(hubot-idotaba, cron ... etc)

        $ npm install

* start hubot

        $ sh hubot.sh start

## Options
* stop hubot

        $ sh hubot.sh stop
